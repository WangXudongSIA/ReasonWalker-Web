ReasonWalker: Reasoning Iterative Vision-and-Language Navigation with Implicit Instructions

Xudong Wang, Baichen Liu, Jiahua Dong, Xi’ai Chen, Yandong Tang, Member, IEEE, Honghai Liu, Fellow, IEEE, Lianqing Liu, Senior Member, IEEE, and Zhi Han, Member, IEEE
Source Code

Comparison of the proposed ReasonWalker with the existing VLN model. a). Traditional VLN model performs navigation with user explicit step-by-step instructions. b). Our ReasonWalker achieves reasoning navigation with user implicit instructions, guided by LLM reasoning scene associations.

Abstract

Existing vision-and-language navigation (VLN) agents often fail to reason users’ implicit goals and cannot leverage past experience in persistent environments. In this paper, we propose ReasonWalker, a novel navigation model designed to enable reasoning-based navigation using implicit instructions over time. To ensure persistent and efficient operation, ReasonWalker constructs and stores explicit scene maps, allowing it to learn scene associations for improved re-navigation in subsequent episodes. To facilitate comprehension and reasoning over implicit instructions, ReasonWalker leverages a large language model to jointly process user instructions, agent observations, and scene maps, generating semantic navigation tokens that guide action prediction. To train ReasonWalker, we propose a new hierarchical learning paradigm, where the model first learns navigation actions and then acquires scene associations for implicit instruction reasoning. Additionally, we also provide a new implicit instruction benchmark to support training and evaluation of reasoning-based navigation task. Extensive experiments demonstrate the effectiveness and superiority of the proposed ReasonWalker.

Illustration of the proposed ReasonWalker pipeline. At time t, the agent observes the RGB image and depth image to perceive the navigation scene, and follows the user's instructions to perform navigation. ReasonWalker consists of two main modules: The Scene Semantic Mapping Module constructs and saves semantic and occupancy maps. The Reasoning Navigation Action Module reasons user instructions with agent observations and scene maps to predict next navigation action.

Statistics of the transformed implicit instruction datasets. The statistics graph contains eight groups of comparison metrics: Average Instruction Length (the average number of words in an instruction), Single Sentence Length (the average number of words in a single sentence instruction), Number of Sentences (the average number of sentences in an instruction), Percentage of Compound Sentences (an instruction consisting of more than one sentence is called to as a compound sentence), Frequency of Explicit Locations (the probability of the occurrence of explicit locations in each instruction), Percentage of Action Words (the probability of the occurrence of an action word in a sentence), SR (success rate), and SPL (success weighted by inverse path length). G1 represents unchanged instructions, and G2 represents our transformed implicit instructions.
